Rede Social Fullstack com Next.js

Projeto de uma rede social desenvolvida com a stack moderna do Next.js App Router, integrando autenticação com Clerk, banco de dados relacional com PostgreSQL via Prisma, e estilização com Tailwind CSS e Shadcn UI. A aplicação utiliza componentes server/client, rotas dinâmicas e estáticas, carregamento otimista, revalidação de dados e manipulação de arquivos com UploadThing. Também implementei Server Actions, formulários interativos e controle robusto de autenticação e autorização. O sistema é performático, escalável e com foco em boas práticas de arquitetura web.
